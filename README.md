# 🚀 BitCode Rennova

> Create stunning motion graphics with the power of editing and coding.

BitCode Rennova is a next-generation motion graphics editor built for creators and developers who want **speed, control, and creativity in one place**.

---

## ✨ What is it?

BitCode Rennova combines:
- 🎬 Timeline-based motion design
- 🧠 Code-driven animation system
- 🎨 Visual editing interface
- ⚡ Real-time preview rendering

Think of it as a hybrid between a motion editor and a creative coding studio.

---

## 🔥 Features (Pre-Alpha)

- 🧱 Layer-based animation system
- ⏱️ Keyframe timeline editor
- 🎯 Real-time canvas preview
- 🎨 Shape, text, and image layers
- 🎞️ Basic export pipeline (WIP)
- 💾 Project save/load system
- 🧩 Extensible engine architecture

---

## 🧠 Vision

BitCode Rennova is part of the BitCode ecosystem, focused on:

- faster creative workflows
- smoother animation control (“liquid motion” 🌊)
- developer-friendly design tools
- modular plugin-based architecture
