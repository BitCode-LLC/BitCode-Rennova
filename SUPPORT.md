# Support

Welcome to the BitCode community! 👋

Need help? We'd love to help you find the right place.

## 💬 GitHub Discussions

We have two discussion spaces:

### 🌐 BitCode Organization Discussions

Use the **BitCode Organization Discussions** for topics that apply to BitCode as a whole, such as:

* General questions
* Community discussions
* Ideas for the BitCode ecosystem
* Feedback about multiple BitCode projects
* Announcements and community conversations

### 🎬 BitCode Rennova Discussions

Use the **BitCode Rennova Discussions** for anything specifically related to BitCode Rennova, including:

* Questions about using Rennova
* Motion graphics and animation workflows
* Tips and tutorials
* Feature suggestions
* Project-specific troubleshooting
* Sharing your creations

Please search existing discussions before creating a new one—your question may already have an answer.

## 🐞 Bug Reports

If you've found a bug, please open a **GitHub Issue** in the repository where the problem occurred instead of creating a discussion.

When reporting a bug, please include:

* Operating system
* Application version
* Steps to reproduce
* Expected behavior
* Actual behavior
* Screenshots or logs (if available)

## 🔒 Security

Please **do not** report security vulnerabilities through GitHub Discussions or public Issues.

Follow the instructions in `SECURITY.md` to report security issues responsibly.

## ❤️ Thank You

Thank you for supporting BitCode! Every question, suggestion, bug report, and discussion helps us improve our projects and build a welcoming community.
